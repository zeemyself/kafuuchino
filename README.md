# React + TypeScript + Vite

This template provides a minimal setup to get React working in Vite with HMR.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Babel](https://babeljs.io/) for Fast Refresh
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/) for Fast Refresh

## Linting and Formatting

This project uses [oxlint](https://github.com/oxc-project/oxc) for linting and [oxfmt](https://github.com/oxc-project/oxc) for formatting.

### Scripts

- `npm run lint`: Run oxlint.
- `npm run format`: Run oxfmt to format code.
- `npm run format:check`: Check code formatting with oxfmt.
